# School Management System

A real-world backend application built using **Java 8, Spring Boot, Hibernate, MySQL**
following.

## Features
- Student Management
- Teacher Management
- Lecture Scheduling with Time Conflict Validation
- Pagination & Sorting
- DTO Mapping using MapStruct
- Auditing (Created By, Created Date)
- Soft Delete
- Spring Security with JWT (Upcoming)

## Tech Stack
- Java 8
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- MapStruct
- Lombok
- Spring Security
- Postman

## Architecture
- CQRS Pattern
- Layered Architecture
- DTO Based APIs

## How to Run
1. Create MySQL database `schooldb`
2. Update `application-example.properties`
3. Run Spring Boot application
4. Test APIs using Postman
